# StudyQuest

StudyQuestは、学習を小さなクエストとして受注し、継続を促すWebアプリです。

## ディレクトリ構成

- `apps/web`: フロントエンド
- `apps/api`: バックエンドAPI
- `docs`: 仕様・設計・開発ルール

## 開発ルール

詳しくは `CONTRIBUTING.md` を参照してください。

## 仕様書

- MVP仕様: `docs/mvp.md`
- 画面仕様: `docs/screens.md`
- API仕様: `docs/api.md`
- DB設計: `docs/db.md`

## セットアップ

```bash
pnpm install
pnpm dev
```
