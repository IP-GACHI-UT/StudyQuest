# StudyQuest

StudyQuestは、学習を小さなクエストとして受注し、継続を促すWebアプリです。

## ディレクトリ構成

- `apps/web`: フロントエンド
- `apps/api`: バックエンドAPI
- `docs`: 仕様・設計・開発ルール
